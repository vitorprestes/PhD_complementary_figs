# PhD_complementary_figs

In this repository, you can find complementary figures and animations of my Ph.D. thesis entitled "***Extracting Physics Parameters from Supernova Burst Neutrinos in DUNE***".

## Ternary diagram animations  

Red dots represent NH case, green dots represent IH case, and blue dots represent the non-oscillated case.  
For CL regions:
 - Yellow regions for NH points
 - Light blue regions for IH points
