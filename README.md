# PhD_complementary_figs

In this repository, you can find complementary figures and animations of my Ph.D. thesis entitled "***Extracting Physics Parameters from Supernova Burst Neutrinos in DUNE***".

Author contact: <vitorprestes@gmail.com>

## Folder description

Here, we have some folders to organize the figures, provided in `.png` files and `.gif` for the animations.

- [neutronization_counts](neutronization_counts) folder contains figures for chapter 4, with results for the **Garching**, **Nakazato**, **WBD**, and **H端depohl** databases.
- [evt_ratio](evt_ratio) folder contains figures for chapter 5, with results for the **WBD**, and **Nakazato** databases.
- [ternary](ternary) folder contains sub-folders for flux diagrams, animations and interaction channel diagrams for **Nakazato**, **WBD**, and **H端depohl** databases.
